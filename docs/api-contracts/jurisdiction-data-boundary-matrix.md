# Sovereign State Domain Isolation: Jurisdiction Data Boundary Matrix

This matrix establishes the legal and technical limits of data visibility. It is mathematically enforced by the API contract Validation Engine and gateway filters. Any attempt to cross-compile or read unauthorized records triggers a high-severity sovereignty infraction alert, immediately blocking the node.

## 2. Enforcement Standards
1. **The Gateway Filter Rule**: Any database query launched from a Joint Operations token targeting raw tables is mathematically blocked and returns `ACCESS_DENIED_SOVEREIGNTY_CHECK`.
2. **The Zero-Knowledge Rule (ZKP)**: Where identity verification is needed across borders, the querying jurisdiction provides only a SHA256 hash or numeric ID. The other jurisdiction returns a boolean state (`true`/`false`) along with its own cryptographic token. Raw name or portrait indexes are NEVER shared.
